# Ascenda Loyalty Test

# Pre-requisites

- Node.js
- NPM

# Installation

## Windows
- Clone the repository
- "cd" into the repository
- Run `npm i` to install the dependencies
- Run `nodemon app.js` to start the server

## Mac
- Clone the repository
- "cd" into the repository
- Run `npm i` to install the dependencies
- Run `nodemon app.js` to start the server
